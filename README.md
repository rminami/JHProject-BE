# JHProject-BE

This is a mock backend for the CS3099 Junior Honours project.

## Installation

This application was tested using

- node v9.9.0
- TypeScript v2.8.1

Dependencies can be installed by running

```sh
$ npm install
```

## Usage

Run 

```sh
$ npm start
```
