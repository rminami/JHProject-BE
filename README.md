# JHProject-BE

A mock backend for the CS3099 Junior Honours Project.

## Installation

This application was tested using

- node v10.4.0
- yarn v1.7.0
- TypeScript v2.8.1

Dependencies can be installed by running

```sh
yarn install
```

## Usage

For development mode, run

```sh
yarn dev
```

Or, for production mode, run

```sh
yarn start
```
